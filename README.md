# Vrinda Store Data Analysis

## Project Overview
This project analyzes sales data of Vrinda Store to understand customer behavior, sales trends, and order distribution.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization
- Data Cleaning

## Analysis Performed
- Monthly sales analysis
- Order count analysis
- Revenue trend analysis

## Key Insights
- Women customers contributed 64% of total orders.
- Amazon generated the highest orders (35%).
- Maharashtra recorded the highest sales (~2.9M).
- 92% of orders were successfully delivered.

## File Included
Vrinda Store Data Analysis.xlsx

## Project Outcome
The analysis helps understand:
-Customer purchasing behavior
-Top-performing sales channels
-Regional sales performance
-Order delivery success rate
